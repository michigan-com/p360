# P360
360 images, using regular JPEGs

## Dependencies
* [Nodejs](http://nodejs.org) (> 0.11.2, for harmony support)
* [Gulp](http://gulpjs.com/)

## Install
### Node stuff
```bash
npm install -g gulp
npm install
```

## Run
Build the stuff
```bash
gulp
```
Run the stuff
```bash
node dist/app.js
```
